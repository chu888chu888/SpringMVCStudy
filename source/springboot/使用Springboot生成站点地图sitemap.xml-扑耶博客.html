使用Springboot生成站点地图sitemap.xml

 2016-05-26 14:30       阅读(481)

sitemap对于搜索引擎优化（SEO）的诸多好处就不说了，网上有很多sitemap生成工具，这类工具有个缺点就是不能和网站正真整合，不能及时的反应网站的更新。所以，我们通过自己写代码来实现sitemap。

这里拿扑耶博客来举例，本博客采用Springboot开发，所以后面的代码都是基于Springboot的。首先，sitemap是个xml文件，我们要通过配置Springboot的返回类型，让它从返回HTML或者JSON改为返回XML。然后就是读取数据库中的文章列表，拼接字符串，生成sitemap.xml啦。下面看具体实现的代码：

package com.poolye.controller;

import com.poolye.model.Article;
import com.poolye.service.ArticleService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestMethod;
import org.springframework.web.bind.annotation.ResponseBody;

import java.text.SimpleDateFormat;
import java.util.List;

/**
 * Created by longge93 on 16/5/26.
 */
@Controller
public class SiteMapController {

    @Autowired
    ArticleService articleService;

    @RequestMapping(value = "/sitemap.xml",method = RequestMethod.GET,produces = {"application/xml"})
    @ResponseBody
    public String spiderMap(){
        String sitemap="<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n" +
                "<urlset xmlns=\"http://www.sitemaps.org/schemas/sitemap/0.9\">";

        List<Article> articleList = articleService.list();

        for(Article article:articleList){
            SimpleDateFormat format = new SimpleDateFormat("yyyy-MM-dd");
            String lastMod = format.format(article.getUpdateTime());

            sitemap+="<url>" +
                    "\n" +
                    "      <loc>http://blog.poolye.com/Article/"+article.getUuid()+".html</loc>" +
                    "\n" +
                    "      <lastmod>"+lastMod+"</lastmod>" +
                    "\n" +
                    "      <changefreq>monthly</changefreq>" +
                    "\n" +
                    "      <priority>0.8</priority>" +
                    "\n" +
                    "   </url>";
        }
        sitemap+="</urlset>";

        return sitemap;
    }
}

打包运行后，你就可以通过 http://blog.poolye.com/sitemap.xml 来访问到sitemap内容了。
